# GithubTrending

## Implement a small web app that is responsive using Reactjs that list the most starred Github repos created in the last 10 days.


### TechStack

- React
- Tailwind CSS
- Shadcn
- Zustand
- Framer Motion
- ReactIcons

# Features
 - Optimized API Calls
 - Infinite Scroll
 - Scroll Top Btn
 - List Item Fade Out Animation
